# PowerShell-File-Management-Utility

## Overview

A PowerShell automation script that scans a folder, counts files, identifies the largest files, and exports file information to a CSV report.

This project was created to practice PowerShell fundamentals including object manipulation, pipelines, sorting, filtering, and CSV report generation.

## Features

* Scan a specified folder
* Count files within the folder
* Identify the 5 largest files
* Display results in the console
* Export file information to a CSV report

## Technologies Used

* PowerShell

## PowerShell Concepts Demonstrated

* Variables
* Objects and Properties
* Pipelines
* Get-ChildItem
* Sort-Object
* Select-Object
* Export-Csv
* Collections and Count Property
